Sinergyrh

Processo de ux/ui design da área completa do candidato do sistema Sinergyrh.

Conheça o Projeto
Área do Candidato

Objetivo:
Reformular o layout do sistema de candidato e melhorar o fluxo de algumas telas.

Pain points e frustrações dos usuários e do negócio:
Layout com design antigo pouco atrativo, usuários relatavam dificuldades de acessar informações por conta de alguns itens do sistema que não eram necessários.

Pesquisa (analise de usuários, pesquisa de competidores, entrevistas, personas, etc):

Através de uma pesquisa feita com alguns usuários, foi descoberto que os mesmos sentiam dificuldade em navegar pelas telas do sistema. Quando entravam para visualizar os requisitos de uma vaga de emprego a tela possuia um banner que tinha o objetivo de levar o usuário de volta para a tela de vagas caso ele não tivesse os requisitos para se candidatar a vaga em questão, porém esse redirecionamento causava confusão nos usuários.

Soluções encontradas:
No novo layout o banner foi retirado e foi adicionado um botão de "Confirmar inscrição" já no topo da vaga e também no final e um botão "Ver outras vagas".

Validação:

Após alguns testes com usuários foi validado que as novas soluções encontradas foram realizadas de forma mais positiva e agradavél.

Sinergyrh

Processo de ux design para a área do candidato do sistema Sinegyrh.

Conheça o Projeto
Área do Candidato

Objetivo:
Reformular o layout do sistema de candidato e melhorar a experiência do usuário.

Pain points e frustrações dos usuários e do negócio:
Dificuldade por parte dos candidatos ao clicar em se inscrever nas vagas onde as empresas pedem alguns campos obrigatórios a mais para finalização da candidatura.

Pesquisa (analise de usuários, pesquisa de competidores, entrevistas, personas, etc):
Para esse projeto foi realizada uma pesquisa de competidores, a fim de entender o que poderia ser melhorado na ferramenta Sinergyrh e também foi feita uma analise de usuários com o time de operação a fim de entender melhor em quais pontos os candidatos sentiam maior dificuldade para se inscrever nas vagas. Através dessa pesquisa foi identificado que no momento em que o usuário tenta se candidatar na vaga é apresentado a ele uma tela informando que é necessário preencher alguns campos obrigatórios solicitados pela empresa contratante e nesse percurso é apresentada uma tela não muito informativa onde o usuário deve escolher entre currículo básico e currículo completo para conseguir se candidatar.

Soluções encontradas:
Na tela de escolha de currículo para preenchimento dos campos apresentar uma tela informativa, explicando que para se candidatar a vaga a empresa pede o preenchimento de mais alguns campos obrigatórios para inscrição. Foi retirado também a escolha entre currículo básico e currículo completo deixando apenas um botão que leva o candidato para os campos exigidos ela empresa.
